Readme - Imvertor

Dit is informatie bij de release met de naam
Imvertor-Imvertor-1.0.0-1-20141001-at-20141022-103548.

Dit bestand is opgebouwd uit de volgende folders:

  • Folder /doc bevat de systeemdocumentatie. Systeem documentatie betreft een rapportage van het aangeboden UML model vanuit het perspectief van de omzetting naar een XML schema.
  • Folder /uml bevat een d.m.v. Enterprise Architect v 2.5 samengesteld HTML report. Dit is informatie die ontleend is aan de applicatie en de applicaties waar het op teruggaat.
  • Folder /xsd bevat het gegenereerde schema, en de schema’s waar dit naar verwijst. Het gegenereerde schema voor de applicatie zelf zit in de folder met de naam "Imvertor". Hierin zijn alle gegenereerde schema’s, per package, opgenomen in eigen folders met de naam van het package. Daarin is het (enige) schema geplaatst. Ook worden voor referentie elementen aparte schema’s opgesteld met de naam X-ref, wanneer van toepassing. 
    xsd/Imvertor/application/v20141001/Imvertor_Application_v1_0_0.xsd
  • Folder /etc bevat (afhankelijk van de status van de applicatie) het samengestelde EAP sjabloon, Imvert informatie over de historie, en Imvert informatie over het UML model. Wie een beetje handig is met XML kan deze bestanden oppikken en er geheel eigen reports op draaien. 
    etc/Imvertor.history.imvert.xml
    etc/Imvertor.model.imvert.xml
    etc/Imvertor.system.imvert.xml

Reacties op de vorm van deze release zijn welkom. Neem contact op met info@armatiek.nl.
